# Assignment 03: Linear Regression

## 📌 Project Title
Linear Regression on Medical Insurance Dataset

## 📊 Dataset Description
- Source: Kaggle (Medical Insurance Dataset)
- Files Used:
  - `Train_Data.csv`
  - `Test_Data.csv`
- Total Records: More than 1000
- Type: Real-world medical insurance data

## 🎯 Objective
To build a Linear Regression model to predict medical insurance charges based on multiple input features and evaluate its performance using appropriate metrics.

## 🎯 Target Variable
- `charges` – Medical insurance cost (output variable)

## 🧾 Input Features
- age
- sex
- bmi
- children
- smoker
- region (encoded)
