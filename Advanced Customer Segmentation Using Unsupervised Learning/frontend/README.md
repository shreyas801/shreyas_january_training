# Customer Segmentation Dashboard - Frontend

A modern, interactive dashboard for the AI-Driven Customer Intelligence System. This frontend provides comprehensive visualization and analysis of customer segmentation results generated by the Python backend.

## 📊 Features

### Overview Dashboard
- **KPI Cards**: Total customers, revenue, segments, and best model performance
- **Cluster Distribution Chart**: Visual breakdown of customer segments
- **Revenue by Segment**: Doughnut chart showing revenue contribution
- **Segment Overview Cards**: Quick view of all customer segments

### Customer Segments
- Detailed segments table with metrics
- Filter by segment type
- Customer count, spending, purchases, income analysis
- Priority badges for each segment

### Business Insights
- Revenue analysis by segment
- Churn risk analysis
- Key findings from clustering
- Priority action items

### Marketing Recommendations
- Segment-specific marketing strategies
- Recommended channels
- Suggested offers
- Budget allocation per segment

### Visualizations
- Age vs Spending Score scatter plot
- Income vs Total Spending comparison
- Purchase frequency distribution
- Recency distribution

### Model Performance
- K-Means, Hierarchical, DBSCAN, GMM comparisons
- Silhouette scores
- Best model identification

## 🚀 Quick Start

### Option 1: Open Directly in Browser
Simply open the `index.html` file in any modern web browser:

```bash
# Windows
start frontend/index.html

# Mac
open frontend/index.html

# Linux
xdg-open frontend/index.html
```

### Option 2: Using a Local Server (Recommended)
For better performance and to avoid CORS issues:

```bash
# Using Python
cd frontend
python -m http.server 8000

# Using Node.js (if installed)
npx serve frontend
```

Then open: `http://localhost:8000`

## 🔧 Customization

### Update Sample Data
Edit `data/data.json` to reflect your actual segmentation results from the Python backend.

### Change Colors
Modify CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    /* ... more variables */
}
```

### Add New Sections
1. Add section HTML in `index.html`
2. Add corresponding styles in `css/styles.css`
3. Add JavaScript logic in `js/app.js`

## 🔗 Integration with Backend

To connect this frontend to your Python backend:

1. **API Integration**: Modify `loadSampleData()` in `js/app.js` to fetch from your API
2. **Data Format**: Ensure backend returns JSON matching `data/data.json` structure
3. **Endpoints Needed**:
   - `/api/segments` - Get segment data
   - `/api/insights` - Get business insights
   - `/api/models` - Get model performance
